Setting up your dataset

The setup form allows you to customize the datasets we'll generate for you, by editing the fields as needed:

➤ How many sets?

    This is the number of variables: if you picture your dataset as a table, the number of sets corresponds to the number of rows.

➤ How many numbers per set?

    This field defines the length of each set - that is, the number of columns in your table.

➤ Min/max digits per number

    These options control the number of digits in each number: for each set, our generator will choose a random length within the range you've submitted and will return the numbers accordingly.

Note: The number of digits will be always be the same within the set.

➤ Currency

    If this option is active, we'll add two decimal points to each number.

Note: This field adds two digits to the min/max range, so there's no need to adjust those.
